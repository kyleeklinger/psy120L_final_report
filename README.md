# psy120L_final_report
Developmental Experiences: Shaping Students' Experience In Academic Environments

  This repository contains the published HTML version of my PSY 120L final reproducible report.

  The study aimed to answer the question: How do students' developmental and academic experiences influence self- efficacy and achievement motivation?

  This report was created using Quarto and R Studio. 

  The report file is index.html.

  The dataset is not included in this repository.
